# Controle de Gastos

Aplicação web para gerenciamento de despesas

## Funcionalidades
- Adicionar/Remover Gastos
- Categorizar
- Cálculo Automático de total
- Dados guardados com LocalStorage
- Gráfico de gastos por categoria (Chart.js)

## Tecnologias
- HTML
- JS
- Chart.js

## Melhorias Futuras
- Filtro por categoria
- Melhoria de Design com CSS

## Como Usar
Abra o arquivo index.html no navegador
